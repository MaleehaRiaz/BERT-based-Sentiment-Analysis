## Project Overview
This project develops a BERT-based deep learning model for sentiment analysis using the IMDB Movie Reviews dataset. The objective is to classify movie reviews into positive and negative sentiments using a pre-trained transformer model.

## Problem Statement
Sentiment analysis is an important Natural Language Processing (NLP) task that identifies whether a text expresses a positive or negative opinion. Manual analysis of thousands of reviews is time-consuming. This project develops a BERT-based sentiment classification model to automatically classify IMDB movie reviews.

## Dataset Description
The IMDB dataset contains:
* 50,000 movie reviews
* Binary sentiment labels (positive/negative)
Each review is used as input text, and sentiment is the target label.
